DigiWatch















A smartwatch is a computerized bracelet. While early models can perform basic tasks, such as calculations, translations, and game-playing, 2010s smartwatches are effectively wearable computers. Many run mobile apps, using a mobile operating system. Some smartwatches function as portable media players, with FM radio and playback of digital audio and video files via a Bluetooth or USB headset. Some models, also called 'watch phones', feature full smartphone capability, and have complete functionality of a typical smartphone
